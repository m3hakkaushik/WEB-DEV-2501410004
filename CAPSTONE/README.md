## Sections Implemented

1. Sticky navbar with search, cart, login, and hamburger menu.

2. Hero banner with sale headline and CTA button.

3. Flash sale countdown strip.

4. Filters sidebar (categories, price, color).

5. Featured categories, product grid, best sellers, testimonials, newsletter signup, and footer.​

## Breakpoints & Layout Techniques

~ Breakpoints: 1024px (tablet adjustments), 768px (mobile sidebar reordering), 640px (mobile navbar, 2-column categories).
~ Techniques: CSS Grid for main content and product grids, Flexbox for navbar and components, mobile-first responsive design with media queries.​

## How to Open Locally

~ Place "capstone.html" and "capstyle.css" in the same folder with required images (e.g., pic1.jpeg, skorts.png). Open capstone.html directly in any browser like Chrome.
~ No server needed; static site. Optional GitHub Pages: Create repo, push files to gh-pages branch for live demo at username.github.io/repo
